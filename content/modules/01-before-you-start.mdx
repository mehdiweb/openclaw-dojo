# Before You Start

## What Is OpenClaw Really?

Let's clear up the confusion:

- **OpenClaw** = The gateway software (connects chat apps to AI)
- **Pi** = The AI agent that actually responds (you'll configure this)
- **Clawd** = Our friendly lobster mascot ü¶û

OpenClaw is a bridge. It lets you talk to AI agents (like Pi) through WhatsApp, Telegram, Discord, or iMessage.

## What You'll Need

Before we start, make sure you have:

### Required:
1. **A VPS or always-on computer** ($6-20/month)
   - DigitalOcean, Hetzner, AWS, or any cloud provider
   - Your laptop won't work (needs to be always on)
   
2. **Node.js version 22 or higher**
   - Check: `node --version`
   - Install: https://nodejs.org
   
3. **Basic terminal comfort**
   - Copy-pasting commands
   - Editing text files
   - SSH into servers

4. **Time: 2-4 hours**
   - First setup takes time
   - Future updates are faster

### For WhatsApp:
- A **separate phone number** (recommended)
  - Your personal number works but isn't ideal
  - Cheap options: Hetzner eSIM, giffgaff, any prepaid SIM
  
### For Telegram:
- A Telegram account
- 5 minutes with @BotFather

### API Keys (for AI):
- Anthropic API key (Claude) - https://console.anthropic.com
- OR OpenAI API key - https://platform.openai.com
- Budget: $5-20/month depending on usage
